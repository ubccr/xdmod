Prerequisites

The tutorial will use the web-based ACCESS XDMoD portal so there is no software needed other than a web browser. The tutorial will also run through an example interactive report that is targeted at end-users.

The conference Wi-Fi will be bandwidth limited, so we encourage you to download the report in advance. The report will be available to download here from 9am EDT on Sunday July 26. Please check back here to download the report before the tutorial starts.
