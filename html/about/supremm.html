The SUPReMM Realm


The SUPReMM realm provides job-level performance data for detailed analysis in the XDMoD user interface or through the Data Analytics Framework. This information is populated by cross-referencing jobs from XDMoD's data warehouse with node-level performance data running on compute nodes.

Node-level performance data are gathered through collection software such as Performance Co-Pilot (PCP), Prometheus, or HPCPerfStats running on compute nodes. These collectors sample node hardware information at a regularly configured interval and provide a wide variety of job performance information including memory, I/O file data, CPU data and network data.

More information about the setup and usage of SUPReMM can be found in the documentation.
